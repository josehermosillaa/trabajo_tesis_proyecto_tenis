# Tennis Tournament Management System

Sistema informático para la gestión operativa de torneos de tenis del Club Estadio Español de Curicó.

## Tecnologías

- Python 3.12
- Django
- Django REST Framework
- Angular
- PostgreSQL
- Bootstrap
- JWT Authentication
